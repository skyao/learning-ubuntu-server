SDKMAN 是一款管理多版本 SDK 的工具，可以实现在多个版本间的快速切换。

> sdkman 支持 macos 和 linux，可以参考：
> 
> https://skyao.io/learning-macos/docs/programing/common/sdkman.html

### 安装sdkman

```bash
$ curl -s "https://get.sdkman.io" | bash
$ curl -s "https://get.sdkman.io" | bash
$ sdk version

SDKMAN 5.15.0
```
